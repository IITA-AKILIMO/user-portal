== Installation ==

### Automatic installation

1. Search for Brave Builder through 'Plugins > Add New' interface.
2. Find the plugin box of Brave Builder  and click on the 'Install Now' button.
3. Then activate the Brave Builder plugin.
4. Create a new Popup: After installing the plugin, a new menu called "Popups" will appear. Visit that page and click the + icon to Create New Popup.


### Manual installation

1. [Download Brave Builder](https://downloads.wordpress.org/plugin/brave-popup-builder.zip)
2. Upload the Brave Builder  through 'Plugins > Add New > Upload' interface or upload bravepopup-free folder to the '/wp-content/plugins/' directory.
3. Activate the Brave Builder  plugin through the 'Plugins' menu in WordPress.
4. Create a new Popup: After installing the plugin, a new menu called "Popups" will appear. Visit that page and click the + icon to Create New Popup.
